# Front-end de um Cadastro de Profissional da Saúde 

### Começo

#### Pré-requisitos
- Necessita Node.js 12 ou mais recente.

#### Instalação

- `git clone https://github.com/matheusroberson/saude.git`
- `cd saude`
- `yarn install`

---

### Iniciando o Projeto

```
yarn serve
```
